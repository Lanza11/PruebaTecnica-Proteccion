# Prueba Técnica - Protección

Este proyecto fue desarrollado como parte de una prueba técnica para Protección. 

## Descripción

El objetivo principal de esta prueba técnica era implementar un **motor de priorización** basado en reglas de negocio específicas. Este motor permite evaluar y ordenar elementos según criterios definidos, facilitando la toma de decisiones automatizada.  

## Características

- Motor de priorización configurable
- Implementación de reglas de negocio
- Sistema de evaluación y ordenamiento automático

## Requisitos de la Prueba Técnica

La prueba técnica solicitaba:
- Diseñar e implementar un motor que pueda priorizar elementos
- Aplicar reglas de negocio específicas para determinar el orden de prioridad
- Crear una solución escalable y mantenible

## Tecnologías Utilizadas

### Backend

### Frontend
